## 🔧 Customization Guide

### Change Clinic Name:
Edit `index.html` line ~20:
```html
<h1>🏥 Your Clinic Name</h1>
```

### Add More Doctors:
Edit `index.html` search for "doctor-card" and add:
```html
<div class="doctor-card">
    <div class="doctor-avatar">👨‍⚕️</div>
    <div class="doctor-info">
        <div class="doctor-name">Dr. Name</div>
        <div class="doctor-specialty">Specialty</div>
        <div class="doctor-experience">X years</div>
    </div>
    <button class="book-btn" onclick="bookWithDoctor('Dr. Name')">Book</button>
</div>
```

### Add More Symptoms:
Edit `index.html` search for `symptomDatabase` and add:
```javascript
'your-symptom': {
    conditions: ['Condition 1', 'Condition 2'],
    advice: 'Your medical advice here',
    severity: 'low' // or 'medium' or 'high'
}
```

### Change Colors:
Edit `index.html` CSS section, search for `#00A8E8` and replace with your color.
